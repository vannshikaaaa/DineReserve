# DineReserve

## Overview
DineReserve is a restaurant reservation application that allows users to easily find and reserve tables at their favorite restaurants. The application consists of both backend and frontend components, providing a seamless experience for users to manage their dining preferences.

## Features
- **User Registration and Authentication**: Users can create accounts and log in to manage their reservations.
- **Restaurant Search**: Users can search for restaurants based on location, cuisine, and availability.
- **Reservation Management**: Users can create, view, and cancel reservations.
- **Admin Panel**: Restaurant managers can manage their listings, including adding new restaurants and updating existing details.

## Technologies Used
- **Frontend**: React, Redux, CSS
- **Backend**: Node.js, Express, MongoDB
- **Authentication**: JWT (JSON Web Tokens)
- **Deployment**: Heroku, AWS (for database)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/vannshikaaaa/DineReserve.git
   ```
2. Install dependencies for backend:
   ```bash
   cd backend
   npm install
   ```
3. Install dependencies for frontend:
   ```bash
   cd frontend
   npm install
   ```
4. Setup the database and environment variables.
5. Run the application:
   - Backend:
     ```bash
     npm start
     ```
   - Frontend:
     ```bash
     npm start
     ```

## Usage
- Follow the on-screen instructions to register and log in.
- Use the search feature to find restaurants and make reservations.
